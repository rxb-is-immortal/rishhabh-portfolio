# Rishabh's Portfolio

A personal portfolio website showcasing cinematic video editing, VFX learning, creative development, and visual projects.

## Overview

This repository contains a static portfolio website built with HTML, CSS, and JavaScript. It includes:

- **Home page** with animated hero section and multimedia presentation
- **Works page** for featured project showcases
- **Info page** for personal introduction and skills
- **Contact page** for direct reach-out
- **Multilingual interface** with language switching support
- **Custom visual effects** and animated UI interactions

## Files and structure

- `index.html` — homepage
- `works.html` — project showcase
- `info.html` — about / skills page
- `contact.html` — contact form and details
- `styles/` — CSS styling, theme, and page-specific styles
- `js/` — page scripts, interactions, animation behavior, and localization logic
- `assets/` — fonts, images, icons, and visual assets

## Technologies

- HTML5
- CSS3
- JavaScript
- GSAP animations
- Lenis smooth scroll
- ScrollTrigger

## Run locally

Since this is a static website, you can open `index.html` directly in the browser.

For a better local experience, run a simple web server from the project root, for example with Python:

```bash
cd E:\PORTFOLIO
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Deployment

This project can be deployed as a static website on GitHub Pages, Netlify, Vercel, or any static hosting service.
